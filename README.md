# ⚡ VLSI Physical Design Internship – Maven Silicon

## 📌 Overview
This repository documents my VLSI Physical Design Internship at Maven Silicon, where I gained hands-on experience in the complete ASIC physical design flow.

## 🏢 Organization
Maven Silicon Softech Pvt. Ltd., Bengaluru

## 📅 Duration
July 2025 – September 2025

## 🚀 Key Work
- Implemented **SPI (Serial Peripheral Interface)** using Qflow
- Performed complete **Place & Route (PnR) flow**
- Conducted:
  - Floorplanning
  - Placement
  - Clock Tree Synthesis (CTS)
  - Routing
- Performed **Static Timing Analysis (STA)** (WNS, TNS)
- Executed **DRC, LVS, ERC checks** for physical verification
- Worked with **MMMC timing scenarios and ECO fixes**
